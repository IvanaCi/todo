## Deployment instructions

    1. Clone repository 
    2. Open terminal at project root directory 
    3. Run `./vendor/bin/sail artisan migrate`
    4. Run `./vendor/bin/sail npm run dev`
    5. The website will be accessible on http://localhost:3000
